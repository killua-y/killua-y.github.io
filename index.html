Mymalloc (C)

• Implement the malloc.c, calloc.c, and free.c function in using mmap
• Design the linked block list structure to manage the memory request by Mymalloc
• When calling Mymalloc, it will go through the linked block list and find a free block. If the free block is larger than the size requested. The remaining space will be split into a new free blocks. If no free block is find, mmap.c will be used to map new space according to the size request. If the size is larger than the page size, then the code will calculate how many page it and map the space of those quantities
• When calling Myfree, the block larger than the page size will be unmaped using munmap.c. For the block less than the page size, the code will check if the adjacent block is free and then merge the free block.
