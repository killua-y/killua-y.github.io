Mymalloc (C)

• Implemented custom memory allocation functions (malloc, calloc, and free) using the mmap system call.
• Designed a linked block list structure to manage memory requests to efficiently allocate and deallocate memory.
• Mymalloc searches the linked block list for a free block that fits the size requested by the user. If the free block is larger than the size requested. The remaining space of that block will be split into a new free blocks. If no block is found, new memory is mapped using mmap according to the size requested. If the size is larger than the page size, the code maps the necessary number of pages.
• Myfree deallocates memory by unmapping blocks larger than the page size using munmap, and merging adjacent free blocks if they are smaller than the page size.
